## Списки

Чтобы добавить ненумеровонные списки, необходимо пункты выделить звездочкой (*).
Например вот так:
* Элимент 1
* Элимент 2
* Элимент 3

Чтобы добавить нумеровонный список, необходимо пункты просто пронумеровать.
Например вот так:
 1. Прервый пункт 
 2. Второй пункт
 3. Третий пункт
